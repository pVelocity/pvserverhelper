### ``logActivity(jsapi, type, source, tag, message)``
Calls the RPM API LogActivity with ``type`` and ``source`` and ``tag`` and ``message``.
- `jsapi` `<Object>`
- `type` `<String>`
- `source` `<String>`
- `tag` `<String>`
- `message` `<String>`

```js
pvh.logActivity(jsapi, 'Info', 'Scenario', 'Create', 'Created new Scenario for ABC');
```
